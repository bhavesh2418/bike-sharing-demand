# Bike Sharing Demand Prediction

A machine learning project to forecast hourly bike rental demand in Washington, DC.

## ✅ Problem
Predict how many bikes will be rented per hour based on:
- datetime
- season
- weather
- temperature
- humidity
- windspeed

## ✅ Goal
Build a regression ML model & optimize using Pipelines + GridSearchCV.

## ✅ Dataset
Source: Kaggle — Bike Sharing Demand

## ✅ Tech Stack
- Python
- Pandas / NumPy
- Scikit-learn
- XGBoost / LightGBM
- Matplotlib / Seaborn
- Git / GitHub

## ✅ Steps
1) EDA
2) Feature Engineering
3) Pipeline + Modeling
4) Hyperparameter Tuning
5) RMSLE Evaluation
6) Deployment
